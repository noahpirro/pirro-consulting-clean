# Pirro Consulting Website

Website for Pirro Consulting - Automation & Digitalization services.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Getting Started

```bash
npm install
npm run dev
```

The development server will start at `http://localhost:8080`.

## Build

```bash
npm run build
npm run preview
```
