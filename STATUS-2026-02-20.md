## 1. PROJEKT-UEBERSICHT

### Was ist Pirro Consulting?
Pirro Consulting ist eine deutsche Unternehmensberatung fuer Automatisierung und Digitalisierung, gefuehrt von Noah Pirro. Die Website ist eine Marketing- und Lead-Generation-Seite, die potenzielle Kunden ueber die Services informiert und zur kostenlosen Potenzialanalyse (Calendly-Buchung) fuehrt.

### Technologie-Stack
- **Frontend:** React 18.3.1 + TypeScript 5.8.3
- **Build-Tool:** Vite 5.4.19 (mit SWC-Plugin)
- **Styling:** Tailwind CSS 3.4.17 + tailwindcss-animate
- **Routing:** React Router DOM 6.30.1
- **Icons:** Lucide React 0.462.0
- **UI-Primitives:** Radix UI (Accordion, Toast, Tooltip, Slot)
- **SEO:** React Helmet Async 2.0.5
- **Toasts:** Sonner 1.7.4
- **Komprimierung:** vite-plugin-compression2 (Gzip + Brotli)
- **Linting:** ESLint 9.32.0 + typescript-eslint
- **Fonts:** Inter (400-700) + Space Grotesk (500-700) via Google Fonts

### WICHTIG: Kein Framer Motion!
Framer Motion wurde vollstaendig aus dem Projekt entfernt (war ~45KB). Alle Animationen nutzen:
- CSS `@keyframes` + `transition` Properties
- Custom `useInView` Hook (IntersectionObserver)
- `AnimatedSection` Wrapper-Komponente
- GPU-beschleunigte Transforms (opacity, transform)
- `prefers-reduced-motion` wird global respektiert
